The Creative Workshop Hub is an Android application designed to help users manage their creative projects efficiently. The app allows users to create, update, and delete projects, each with detailed information such as project name, description, type, start date, and end date. Users can also share project details and set notifications for project start and end dates. The application is built using Java and Kotlin, with Android Studio as the primary development environment.
